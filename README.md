# 🌸 青春恋曲 - 像素风恋爱 RPG

一段从初中到工作的浪漫爱情故事，讲述了男主和女主从相识、相知到相爱的漫长旅程。

## 📖 故事章节

1. **第一章：初中时光** - 新同桌，发现小学的缘分
2. **第二章：初二的告白** - 热烈追求，第一次告白被拒，默默陪伴
3. **第三章：暑假夜市** - 和闺蜜折幸运星，摆摊，他买走三瓶
4. **第四章：高中重逢** - 同一高中，无人机礼物，化学补习
5. **第五章：大学甜蜜** - 一起返校，疫情网课，接受告白，甜蜜时光
6. **第六章：研究生与求婚** - 远方读研，毕业旅行，玄武湖求婚
7. **第七章：工作与家庭** - 惊喜投喂，见双方父母，永远幸福

## 🎮 如何运行

1. 直接在浏览器中打开 `index.html` 文件即可游玩
2. 或者启动一个本地服务器：
   ```bash
   python3 -m http.server 8000
   ```
   然后访问 `http://localhost:8000`

## 🎯 游戏操作

- **点击屏幕**或按**空格/回车**：继续对话
- **ESC**：打开/关闭菜单
- 菜单中可以：继续游戏、选择章节、查看花絮、保存进度

## 💖 游戏特色

- 完全使用原生 HTML5 Canvas 开发，无需任何外部依赖
- 精美的像素艺术风格
- 完整的 7 个章节，24 个花絮
- 存档和读档功能
- 章节选择和花絮画廊
- 充满爱意和幸福感的剧情描述
