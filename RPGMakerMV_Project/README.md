# 青春恋曲 - RPG Maker MV 项目

## 项目概述

这是一个从 HTML5 版本的恋爱游戏改编而来的 RPG Maker MV 项目，保留了所有7个章节的对话、场景和关卡。美术资源留给您自由发挥。

## 章节列表

1. **第一章：初中时光** - 新同桌、小学的缘分
2. **第二章：手机里的告白** - 手机告白、学习为重
3. **第三章：幸运星的约定** - 夜市摆摊、三瓶幸运星
4. **第四章：重逢在高中** - 同一所高中、无人机礼物
5. **第五章：大学的甜蜜** - 接受告白、美食时光
6. **第六章：研究生与求婚** - 毕业旅行、玄武湖求婚
7. **第七章：见家长与未来** - 见双方父母、永远幸福

## 使用说明

### 导入到 RPG Maker MV

1. 打开 RPG Maker MV
2. 选择「文件」→「新建项目」，创建一个新项目
3. 保存并关闭项目
4. 将 `RPGMakerMV_Project/data/` 目录下的所有 JSON 文件复制到您的新项目的 `data/` 目录中，替换原有文件
5. 重新用 RPG Maker MV 打开项目

### 自定义内容

由于本项目只保留了对话和关卡结构，您需要自己添加：

1. **美术资源**
   - 在 `img/` 目录下添加角色图、脸图、图块集、背景图等
   - 在 `audio/` 目录下添加背景音乐和音效

2. **地图设计**
   - 使用 RPG Maker MV 的地图编辑器设计每个章节的场景
   - 添加相应的图块和装饰

3. **事件调整**
   - 可以在 RPG Maker MV 中进一步调整对话事件
   - 添加更多交互元素和选项

4. **角色设置**
   - 完善角色的脸图和行走图
   - 设置合适的角色属性

### 游戏流程

每个地图都包含该章节的全部对话，完成后会自动跳转到下一章。玩家可以通过点击继续对话。

## 注意事项

- 本项目是基于原始 HTML5 游戏的完整对话内容转换而来
- 所有对话、场景结构都已保留
- 美术资源完全留白，留给您自由创作
- 建议在导入后先测试游戏流程，再开始添加美术资源
